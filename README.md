# Finance Premium

Aplicativo financeiro que guarda os lançamentos somente no armazenamento do navegador do próprio celular.

## Instalar

Publique esta pasta em um site com HTTPS. No Android, abra no Chrome e escolha **Instalar app**. No iPhone, abra no Safari, toque em **Compartilhar** e escolha **Adicionar à Tela de Início**.

Os dados não são enviados para nenhum servidor. Limpar os dados do navegador ou desinstalar o site pode apagar os lançamentos; use o mesmo aparelho e navegador para mantê-los.
